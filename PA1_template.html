2. What is mean total number of steps taken per day?

First, we calculate the total number of steps per day.

totalstepbydate <- data %>%
        filter(!is.na(steps)) %>%
        group_by(date) %>%
        summarise(totalstep=sum(steps,na.rm=TRUE))

hist(totalstepbydate$totalstep,breaks=30,xlab="Total steps per day", main="Distribution of daily steps")

2.2. Calculate and report the mean and median total number of steps taken per day

First we calculate the mean and median value of daily steps:

meantotalstep <- as.integer(round(mean(totalstepbydate$totalstep)))
mediantotalstep <- median(totalstepbydate$totalstep)
Therefore, the mean value of daily total steps is 10766, the median value is 10765.

4. Imputing missing values

_4.1. Calculate the total number of missing values. _

totalna <- sum(is.na(data$steps))
Therefore, the total number of missing values is 2304.

_4.2-4.3. Fill in the missing values, here I am using the mean of the 5-min interval as the estimate. Create another dataset with the missing values filled in. _

data2 <- data %>%
        merge(avgintervalstep,by.x="interval",all=TRUE) %>%
        filter(is.na(steps)) %>%
        mutate(steps=avgstep) %>%
        select(steps,date,interval) %>%
        rbind_list(data[!is.na(data$steps),]) %>%
        arrange(date)

5. Are there differences in activity patterns between weekdays and weekends?

_5.1. Create new factor variable indicating it's weekday or weekend. _

data2 <- data2 %>%
        mutate(wd=weekdays(date),wday="")
data2$wday[data2$wd %in% c("Saturday","Sunday")] <- "weekend"
data2$wday[!data2$wd %in% c("Saturday","Sunday")] <- "weekday"
data2 <- data2 %>%
        select(-wd) %>%
        mutate(wday=as.factor(wday))
